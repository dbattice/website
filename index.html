Basic equation for vibrations: m*x''+c*x'+k*x=0 m=mass, c=damper, k= spring constant, x=x(t), Since this is a second order differential equation it will have one of three solutions.

Overdampened case when c>2*sqrt(m*k)

x(t)=a*e(-c*t)+b*e(-d*t)

Perfectly dampened case when c=2*sqrt(m*k):

x(t)=a*e(-m*t)+b*e(-m*t)*t

Underdampened case when c<2*sqrt(m*k):

x(t)=a*e(-m*t)*cos(wt+phi)+b*e(-m*t)*sin(wt+phi)
